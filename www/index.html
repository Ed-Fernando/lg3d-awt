Welcome to Project Looking Glass Abstract Windowing Toolkit (AWT) Project!

This project is an Abstract Windowing Toolkit (AWT) for Project Looking Glass. The primary goal is to be able to run unmodified AWT/Swing applications and applets inside the Looking Glass Desktop. An attempt has been made to make this implementation generic enough so that it can be used to run unmodified AWT/Swing applications inside any Java container (an already running JVM)

WARNING! The code in the CVS repository for this project is currently incomplete and does not build properly. Please check back in a few weeks for a initial working implementation integrated with the Looking Glass core.
